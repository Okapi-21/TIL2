## Linux コマンド

#### 所有者権限の切り替え

##### chmod 

　ファイルやディレクトリの rwx などのアクセス権（パーミッション）を変更するためのコマンド（所有者：グループ：その他）で管理可能

``` 
chmod 755 filename
```



##### chown

　ファイルやディレクトリのユーザー・グループの所有者を変更

```
chown apache:apache myfile.txt

#再帰的
chown  -R apache:apache myfile/dir
```



##### setfacl

　setfal（セット・エフ・シー・エル）は**Linuxのファイルやディレクトリに対して、「追加のアクセス権」を設定するコマンド**のこと。
普通のLinuxの権限は「所有者」「グループ」「その他」だけであるが、ACL（Access Control List：アクセス制限リスト）を使用すると「このユーザーにだけ特別に書き込みを許可したい」といった細かな権限管理が可能となる。

ex ) 

```
sudo setfacl -R -m u:opc:rwx storage bootstrap/cache
```

> [!TIP]
>
> -  -m ： ACLを追加・上書きで設定する 、という意味
> - u : opc : rwx ：opcユーザーに対して r（読み込み）・w（書き込み）・x（実行）を許可する（ ユーザー：ユーザー名：権限）
> - d : opc : rwx ：意味としては上記とほぼ等しいけれど、u ではなく d （デフォルト）なので 、新しく作られるファイル・ディレクトリに関しても
>   　　　　　　　自動的に管理を渡すように誘導することができる
