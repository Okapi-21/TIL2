## エージェント

------

AIエージェントは、大規模言語モデル（LLM）を中核とした自律システムであり、複雑なタスクを効率的にこなす力を持つ。
基本的にエージェントは以下のプロセスに従って処理を進めていく。

①ユーザー入力：プロセスの起点は、ユーザーからの指示やクエリ、または多様なデータ入力である。
②目標設定：ユーザーの入力を受け、AIエージェントは具体的な目標を定める。ここでタスクの方向性が決まる。
③大規模言語モデル（LLM）：エージェントの頭脳とも言える部分。高度な自然言語処理能力を持ち、入力情報を理解・分析し、行動計画を立案する。
④ツール選択：LLMの判断に基づき、タスク達成に最適なルールやアプローチを選択。必要に応じてAPIや外部データベースなどを使い分ける。
⑤タスク実行：選ばれたツールを使って実際の作業を進める。外部サービスとの連携などを通して、情報収取・処理を行う。
⑥モニタリング：作業の進捗を常に監視し、状況に応じて先着を修正する。これにより、突発的な問題にも柔軟に対応することができる。
　　　　　　　　モニタリング結果をLLMに戻して、楽手や改善を継続的に行う。

以上のサイクルにより、AIエージェントは自律的に複雑なタスクを進め、人間の指示を理解しながら状況に合わせて方針を柔軟に変えることができる。
カスタマーサポートやデータ分析、研究支援、プロジェクト管理など、応用範囲は広大と言える。

AIエージェントは従来のチャットボットと異なり、目標を設定し、複数のツールを組み合わせながら、自律的に問題解決に取り組むことができる。
「時刻確認」「情報検索」「計算処理」など、複数のツールを状況に応じて使い分けることで、より正確な問題解決が可能になる。また、特定のナレッジを組み込むことで、専門的な判断も可能ということになる。特に、具体的な数値計算との組み合わせとなると、実用的な場面での活躍が期待できる。

また、使用するモデルによっては、ビジョン（画像解析）とドキュメント（文書解析）を使用することができる。
（つまり、画像や文書を提示して、その内容について尋ねることが可能になる）



プロンプト

```
あなたが考える場合はステップバイステップで独り言で考えます。
深く深く考えます。それらの独り言を明示的に出力しつつ、それを参照し、検証し、さらに深く考えます。
```

```
あなたは博識なアシスタントです。難しい質問に対して、詳細かつ正確に答えてください。
必要に応じて、専門亭な知識や最新の情報を含めて回答して下さい。
なお、できるだけ詳しい内容が必要なので出力文字数を気にしないでしゅつ食して下さい。
```
